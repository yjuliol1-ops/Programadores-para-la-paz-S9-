Sistema operativo: Es el software que administra el funcionamiento del computador y permite ejecutar programas. También controla los recursos del sistema, como la memoria, el almacenamiento y los dispositivos.

Node.js: Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y desarrollar aplicaciones web.

Aplicación server.js: Es el archivo donde se escribe el código del servidor. Define las rutas, las respuestas y el comportamiento de la aplicación cuando recibe solicitudes.

Puerto: Es un canal de comunicación que utiliza el servidor para recibir conexiones. Permite que los usuarios accedan a la aplicación mediante una dirección como http://localhost:3000.

Logs: Son registros que almacenan información sobre la actividad y los eventos del servidor. Ayudan a detectar errores, monitorear el funcionamiento y solucionar problemas.
